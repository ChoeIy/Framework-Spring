# 🎮 Mini Jogo com Spring Boot

Projeto simples feito com **Spring Boot** + HTML + JavaScript, com foco em aprendizado.

---

## 🚀 Funcionalidade

A aplicação exibe um quadrado na tela que pode ser movimentado usando o teclado:

* ⬅️ ➡️ ⬆️ ⬇️

O movimento é contínuo e limitado dentro da tela.

---

## 🛠️ Tecnologias

* Java 17
* Spring Boot
* HTML / CSS
* JavaScript

---

## 📂 Estrutura

```
src/main/java/com/example/demo
 ├── DemoApplication.java
 └── controller
      └── HomeController.java

src/main/resources/templates
 └── index.html
```

---

## ▶️ Como executar

No terminal:

```
.\mvnw spring-boot:run
```

Depois acesse no navegador:

```
http://localhost:8080/
```

Este projeto foi criado para praticar:
